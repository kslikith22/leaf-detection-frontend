
## Setup for local development
- Install `yarn`, `nodejs` (Refer Miscellaneous)



- Create `.env` files for client

REACT_APP_API_URL = http://localhost:8080/predict  (There is seperate repo to run backend , ping me for link)


- Setup project
```bash
yarn install
yarn run start
```
